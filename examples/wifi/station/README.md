# station

## 功能概述
本例演示如何使用 Wi-Fi 的 station 功能。

## 环境要求

1. 需要有一个能正常连接的AP。
2. 在 menuconfig 中配置 WiFi SSID 和 WiFi Password。

## 编译和烧录

示例位置：`examples/wifi/station`

编译、烧录等操作请参考：[快速入门](https://doc.winnermicro.net/w800/zh_CN/latest/get_started/index.html)
