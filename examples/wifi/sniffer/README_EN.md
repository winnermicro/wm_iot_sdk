# snffer

## Overview
This example demonstrates the Wi-Fi snffer functionality . Wi-Fi snffer can capture WiFi packets during abnormal communication. This can be useful in network analysis, security testing, and other applications where Wi-Fi traffic needs to be monitored and analyzed.

## Requirements
1. Just flashing the program code normally

## Building and Flashing

Example Locationï¼š `examples/wifi/snffer`

For compilation, flashing, and other operations, please refer to: [Quick Start Guide](https://doc.winnermicro.net/w800/en/latest/get_started/index.html)
