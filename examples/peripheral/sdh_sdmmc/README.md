# SDH_SDMMC

## 功能概述

此应用程序是使用 `WM IoT SDK` 进行 `SDH_MMC` SD卡读写功能的全面示例。它演示了如何使用 `WM IoT SDK` 中的 `wm_drv_sdh_sdmmc` 驱动 API 进行 SD 卡的读写操作。示例包括单块和多块数据的读写，并验证数据的一致性。

## 环境要求

需要支持 SD 卡的硬件环境，并插上 SD 卡


## 编译和烧录

示例位置：`examples\peripheral\sdh_sdmmc`

编译、烧录等操作请参考：[快速入门](https://doc.winnermicro.net/w800/zh_CN/latest/get_started/index.html)
