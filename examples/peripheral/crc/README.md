# CRC demo

## 功能概述

启动一个 FreeRTOS 任务，运行一次 crc demo。

crc demo 中遍历了 W800 支持的 crc 类型，以及是否对输入输出数据进行反向处理。

## 环境要求

含有 w80x 的板子。

## 编译和烧录

示例位置：`examples/peripheral/crc`

编译、烧录等操作请参考：[get started](https://doc.winnermicro.net/w800/en/latest/get_started/index.html)
