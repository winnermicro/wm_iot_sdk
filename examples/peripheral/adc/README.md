# adc 示例

## 功能概述

此应用程序启动后执行下面如下操作：

1. 初始化 adc
2. 启动线程，在线程中去读芯片内部温度，芯片电压以及通道 0 的数据

## 环境要求

请务必注意，adc 的通道和 io 有对应关系，配置时候请参考 wm_adc_channel_t 中的定义

## 编译和烧录

示例位置：`examples/peripheral/adc`

编译、烧录等操作请参考：[快速入门](https://doc.winnermicro.net/w800/zh_CN/latest/get_started/index.html)
