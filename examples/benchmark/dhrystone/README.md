# Dhrystone

## 功能概述

Dhrystone 用于测试处理器性能。

## 环境要求

无。

## 编译和烧录

示例位置：`examples/benchmark/dhrystone`

编译、烧录等操作请参考：[快速入门](https://doc.winnermicro.net/w800/zh_CN/2.2-beta.2/get_started/index.html)
