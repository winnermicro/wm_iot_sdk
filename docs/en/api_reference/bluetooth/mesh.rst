.. _bluetooth_mesh:

Bluetooth Mesh 
######################

Bluetooth Mesh has implemented the
`Bluetooth Mesh Protocol Specification v1.1 <https://www.bluetooth.com/specifications/specs/mesh-protocol/>`_.
Compared to Mesh v1.0, version 1.1 adds secure wireless multi-hop communication functionality.

For more information on Bluetooth Mesh, please refer to the
`Bluetooth SIG Website <https://www.bluetooth.com/bluetooth-resources/?tags=mesh>`_.
