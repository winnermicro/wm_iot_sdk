.. _bluetooth_mesh:

蓝牙 Mesh 
######################

蓝牙 Mesh 实现了
`Bluetooth Mesh Protocol Specification v1.1 <https://www.bluetooth.com/specifications/specs/mesh-protocol/>`_. 相对于 Mesh v1.0，v1.1版本增加了安全的无线多跳通信功能。

有关 Bluetooth Mesh 的更多信息请参考
`Bluetooth SIG Website <https://www.bluetooth.com/bluetooth-resources/?tags=mesh>`_.
