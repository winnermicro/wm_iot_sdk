VS Code 调试
^^^^^^^^^^^^^^^^^^^^^^^

在连接 cklink 调试器之后，打开VS Code

在工程文件上右键选择 Debug 命令启动调试

.. figure:: ../../_static/debug/cklink_debug.png
   :align: center

也可以在底部状态栏选择 WM IOT SDK 工具中的 Debug 命令启动调试。
