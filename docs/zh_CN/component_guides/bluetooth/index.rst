Bluetooth
#########

本章节包含 WM IoT SDK 蓝牙子系统信息，您可以使用此信息来了解蓝牙协议栈，蓝牙底层控制器的操作及其实现方式。

WM IoT SDK 蓝牙子系统包含了完整的从应用程序到射频收发的 Bluetooth Low Energy（BLE）应用和部分的传统蓝牙应用。
